# BattleShits – The Crappiest Naval Battle

A Battleship game implementation built with **pure JavaScript**, **Test-Driven Development (TDD)** using **Jest**, and following The Odin Project's JavaScript curriculum.

Live Demo:  
 https://fico27.github.io/Battleship/

> "BattleShits: The Crappiest Naval Battle"

## Features

- **Player vs Computer** gameplay
- **Ship placement phase**:
  - Drag-and-drop or click-to-place ships (horizontal/vertical)
  - "Rotate" button to change orientation
  - "Randomize" button for quick random placement
  - "Reset" to start over
- Turn-based attacking: click enemy grid squares
- Visual feedback: hits (red), misses (blue/gray), sunk ships
- Computer makes random legal moves (no smart AI – keeps it fair and simple)
- Game ends when all ships of one side are sunk → winner announcement
- Fully tested core logic with Jest

## Technologies Used

- **Language**: JavaScript
- **Testing**: Jest
- **Build/Deploy**: GitHub Pages
- **Styling**: Vanilla CSS (minimal & playful)
- **No external frameworks/libraries** for game logic (pure JS + TDD)
